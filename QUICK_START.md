### 1. Download Everything
Click the download button for the fjordhjem folder and save it to your computer.

### 2. Update Your Links
Before deploying, open `index.html` in any text editor and update:

**Line ~210**: Replace `#` with your actual Airbnb listing URL
```html
<a href="YOUR_AIRBNB_URL_HERE" class="btn-booking airbnb"...
```

**Line ~215**: Replace `#` with your actual VRBO listing URL
```html
<a href="YOUR_VRBO_URL_HERE" class="btn-booking vrbo"...
```

**Line ~225**: Replace with your real email
```html
<a href="mailto:your-real-email@example.com"...
```

### 3. Create GitHub Repository
1. Go to https://github.com/new
2. Name it: `fjordhjem` (or whatever you prefer)
3. Make it Public
4. Click "Create repository"

### 4. Upload Files to GitHub
The easiest way:
1. On your new repository page, click "uploading an existing file"
2. Drag ALL files from your fjordhjem folder
3. Click "Commit changes"

### 5. Deploy to Cloudflare Pages
1. Go to https://pages.cloudflare.com/
2. Sign up/in (it's free)
3. Click "Create a project"
4. Connect to Git â†’ Select your GitHub repo
5. Click "Save and Deploy"

Your site will be live at: `https://fjordhjem.pages.dev` in ~2 minutes

### 6. Connect Your Domain (fjordhjem.com)
Once live:
1. In Cloudflare Pages, click "Custom domains"
2. Add `fjordhjem.com`
3. Follow the DNS instructions
4. Wait a few minutes for it to propagate
